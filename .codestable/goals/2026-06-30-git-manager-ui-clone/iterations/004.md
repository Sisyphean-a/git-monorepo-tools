# Iteration 004

## 目标

补齐真实 diff 预览、空按钮本地能力、设置持久化和自动扫描。

## 实现

- 新增前端设置模型 `src/app/settings.ts`，把 AI 提交、Git 行为、扫描目录与自定义分类持久化到 `localStorage`。
- 扩展本地 Vite API：
  - `scripts/sync-real-data.mjs` 支持按设置扫描额外目录、按文件返回真实 diff、读取仓库日志，并让 pull/push 使用设置中的策略。
  - `scripts/vite-git-api.mjs` 新增真实 diff、日志、打开文件夹、打开终端、打开冲突处理器、系统目录选择器接口。
  - `scripts/local-system.mjs` 提供 Windows 本地能力封装。
- UI 接线：
  - `Workspace` 接入真实 diff 预览面板，并把文件夹、终端、冲突处理器、日志按钮接到本地能力。
  - `PullAllDrawer` 接入打开、重试、日志、复制报告。
  - `Sidebar` 的新增入口接入弹层，自动扫描开关改为真实设置驱动。
  - `SettingsModal` 改为受控表单，支持保存、添加扫描目录、展示自定义分类。
  - 新增 `AddRepoMenu` 与 `LogViewerModal`。

## 验证

- 执行 `npm run build` 通过。
- 构建过程中 `npm run sync:data` 成功生成最新 `src/app/data.ts`。

## 结果

本轮已完成：

- 真实 diff 预览。
- 空按钮的本地能力接线。
- 设置持久化。
- 自动扫描按设置生效。
