# Git 状态与刷新

这里定义应用读取 Git 状态、展示详情和执行操作时使用的术语。

## Language

**仓库快照**：
某一时刻用于展示仓库状态的基础数据，包含状态、变更列表和远端关系，但不包含可延后读取的提交详情。
_Avoid_: 全量详情、完整快照

**交互操作**：
用户发起且需要立即反馈的仓库操作，例如暂存、取消暂存、提交和单仓本地刷新。
_Avoid_: 前台任务、用户任务

**后台刷新**：
自动扫描或手动全局刷新触发的非交互式状态读取，可包含远端同步。
_Avoid_: 全局任务、轮询任务

**按需详情**：
只在用户打开对应界面或展开对象后才读取的提交、目录或文件信息。
_Avoid_: 延迟数据、懒数据
